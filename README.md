# MyKB

MyKB 是一些知识和随笔的记录，使用 Obsidian 编写及阅读。

## 开始

在开始之前，请确保已经安装了这些依赖：

* python
* make
* texlive

然后可以执行下面的命令来构建出该项目：

```
$ git clone https://github.com/418-teapot/MyKB.git
$ cd MyKB
$ python make.py
```

获得帮助：

```
$ python make.py -h
```

清理临时文件：

```
$ python make.py --clean
```
